# 🚗 qb-tiretheft (FiveM Car Tires Theft Script)

A lightweight, optimized, and immersive tire theft script for FiveM servers. This script allows players to realistically jack up vehicles, remove their wheels using animations, and harvest them for items or sell them on the blackmarket. Completely standalone/synced and **100% FREE**!
